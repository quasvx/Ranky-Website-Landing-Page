 [ENGLISH] 

This repository contains the source code for the **Official Ranky Landing Page**. Designed with a modern and minimalist aesthetic, it serves as the main gateway for users to download the mod, watch video demos, and join the community.

### ✨ SITE FEATURES
 High-Impact Interface
Glassmorphism Design: Modern blur and glass effects optimized for any device.
Bilingual Support: Intelligent translation system (ES/EN) that automatically detects the browser language.
Performance Optimized: Lightweight Vanilla JavaScript structure for instant loading speeds.

⚙️ Integrations
YouTube Preview: Built-in dynamic video player system to view content without leaving the site.
SEO/OG Filters: Advanced metadata configuration ensuring a professional look on Discord, WhatsApp, and Twitter.
Responsive Design: Full adaptability for mobile, tablet, and desktop.
